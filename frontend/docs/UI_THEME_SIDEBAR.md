# Тема и боковое меню

## Тема (светлая / тёмная / системная)

- **Светлая** — светлый интерфейс.
- **Тёмная** — тёмный интерфейс.
- **Системная** — автоматически по настройкам ОС (`prefers-color-scheme`).

Выбор темы сохраняется в `localStorage` (ключ `getsale-theme`) и применяется при следующем заходе. Чтобы не было мерцания при загрузке, в `<head>` выполняется небольшой скрипт, который до отрисовки страницы выставляет класс `light` или `dark` на `<html>`.

Переключатель темы и переключатель языка (EN / RU) находятся в **правом верхнем углу** шапки контента: слева заголовок страницы, справа — блок темы (три иконки: Солнце / Луна / Монитор) и блок языка (кнопки EN, RU).

## Боковое меню (сворачивание)

- По умолчанию меню **свёрнуто**: узкая полоска с иконками (ширина 4rem / 16).
- Кнопка разворота — иконка «панель влево» в шапке сайдбара; при развёрнутом меню — иконка «закрыть панель» справа от логотипа.
- В свёрнутом виде у пунктов меню показывается подсказка (`title`) при наведении.
- Состояние (свёрнуто/развёрнуто) сохраняется в `localStorage` (ключ `getsale-layout`).

Ширина: свёрнуто — `w-16`, развёрнуто — `w-64`. Контент справа сдвигается (`ml-16` / `ml-64`) с плавным переходом.

## Выравнивание

- Шапка сайдбара и шапка контента имеют одинаковую высоту (`h-14` / 3.5rem) и общую нижнюю границу (`border-b`), чтобы линия разделения совпадала.
- Отступы: сайдбар `px-3` в зоне логотипа, контент `px-6` в шапке и в `main`.

## Заголовок страницы

В шапке контента слева отображается название текущего раздела (Главная, CRM, Воронка, Сообщения и т.д.) в зависимости от пути; справа — переключатели темы и языка.

## Семантические цвета

В layout используются токены темы Tailwind: `bg-background`, `text-foreground`, `bg-card`, `border-border`, `text-muted-foreground`, `text-primary`, `bg-accent`, `text-destructive`, чтобы интерфейс корректно менялся при переключении светлой/тёмной темы.
